# SimpleTodoWithSwiftData
## はじめに
SwiftDataの練習がてら作成したシンプルなTodoアプリです。
現在は基本的なCRUD処理、Relation周りのみの実装になっています。

## 環境
- Xcode version15.0
- Swift5
- macOS Sonoma14.1

## サンプルアプリの機能
- Todoの追加
- 完了状態での絞り込み
- Todoのカテゴリ作成
- カテゴリ削除時に該当カテゴリに属しているTodoも削除される
